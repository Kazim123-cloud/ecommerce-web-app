# 🛍️ ShopNest – E-Commerce Web Application

> **Thiranex Internship – Task 3** | Full-Stack Web Development Intern

A fully functional e-commerce web application built with vanilla HTML, CSS, and JavaScript. Includes product catalog, cart system, user authentication, order management, and admin panel.

---

## 🚀 Live Demo

Open `index.html` in your browser — no server needed!

---

## ✨ Features

### 👤 User Features
- **Product Catalog** – Browse 8+ products across Electronics, Fashion, Home & Books
- **Search & Filter** – Filter by category, sort by price/rating, live search
- **Product Detail** – Detailed view with features, pricing, quantity selector
- **Shopping Cart** – Slide-out cart with qty controls, real-time total
- **User Auth** – Register, Login, Logout with role-based access
- **Checkout** – Shipping form + payment method selection (COD / UPI / Card)
- **Order Tracking** – View all placed orders with status

### 👑 Admin Features
- **Dashboard Stats** – Total products, orders, revenue, users
- **Product Table** – Full product inventory view
- **Order Management** – View all orders + update status (Processing → Shipped → Delivered)
